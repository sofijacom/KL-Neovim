# KL Neovim

## Using

Mostly Notes to self.


Backup current neovim configuration files.

```sh
# required
mv ~/.config/nvim{,.bak}

# optional but recommended
mv ~/.local/share/nvim{,.bak}
mv ~/.local/state/nvim{,.bak}
mv ~/.cache/nvim{,.bak}
```

Clone repository.

```sh
git clone https://github.com/alecrissf/nvim ~/.config/nvim
```
